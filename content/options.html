containerStyle

  • Type: Object
  • Default: { position: 'relative' }

CSS styles that are applied to the container element.

columnWidth

  • Type: Number, Element, or Selector String

The width of a column of a horizontal grid. When set, Packery will align item elements horizontally to this grid.

``` js { "columnWidth": 60 } ```

If set to an Element or Selector String, Packery will use the width of that element. See Element sizing.

``` html
...
``` ``` css #column-width-demo2 .grid-sizer { width: 20%; } ``` ``` js { "columnWidth": ".grid-sizer", "itemSelector": ".item" } ```

gutter

  • Type: Number, Element, or Selector String

The space between item elements, both vertically and horizontally.

``` js { "gutter": 10 } ```

If set to an Element or Selector String, Packery will use the width of that element. See Element sizing.

``` html
...
``` ``` css #gutter-demo2 .gutter-sizer { width: 3%; } ``` ``` js { "columnWidth": ".gutter-sizer", "itemSelector": ".item" } ```

isResizable

  • Type: Boolean
  • Default: true

Triggers layout when browser window is resized.

itemSelector

  • Type: Selector String

Specifies which child elements to be used as item elements. Setting itemSelector is always recommended. itemSelector is useful to exclude sizing elements.

placedElements

  • Type: Element or Array of Elements

Specifies which elements are placed within the layout. These are special layout elements which will not be laid out by Packery. Rather, Packery will layout item elements around placed elements.

placedElements places elements only when the Packery instance is first initialized. You can place additional elements afterwards with the place method.

``` html
....
``` ``` js { "itemSelector": ".item", "placedElements": ".placed" } ``` ``` css /* position placed elements with CSS */ #placed-elements-demo { position: relative; } #placed-elements-demo .placed { position: absolute; width: 30%; height: 60px; background: red; border: 4px solid #C00; } #placed-elements-demo .placed1 { left: 20%; top: 0; } #placed-elements-demo .placed2 { right: 10%; top: 80px; } ```

rowHeight

  • Type: Number, Element, or Selector String

Height of a row of a vertical grid. When set, Packery will align item elements vertically to this grid.

``` js { "rowHeight": 60 } ```

If set to an Element or Selector String, Packery will use the height of that element. See Element sizing.

``` html
...
``` ``` css #row-height-demo2 .grid-sizer { height: 60px; } ``` ``` js { "columnWidth": ".grid-sizer", "itemSelector": ".item" } ```

transitionDuration

  • Type: String
  • Default: "0.4s"

The time duration of transitions for item elements.

Element sizing

For the sizing options columnWidth, rowHeight, and gutter, you may set these options to an Element or String, in addition to a Number.

With an Element, Packery will use its width or height to set the value of the related property.

``` js var container = document.querySelector('#container'); var pckry = new Packery( container, { itemSelector: '.item', columnWidth: container.querySelector('.column-width-sizer') }); ```

With a String, Packery will use the string as a selector to get the first matching element within the container element. The size of that element is then used.

``` js columnWidth: '.column-width-sizer' ```

This allows you to control the size of the Packery layout just with your CSS. This is useful for responsive layouts, keeping control within CSS so you can rely on media queries.

``` css .column-width-sizer { width: 20%; } ```

If you are using element sizing, be sure to set itemSelector as well, so the sizing element does not get used in the layout.
