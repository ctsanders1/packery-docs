Events

Packery is an Event Emitter. You can bind listeners to events.

```js var pckry = new Packery( elem ); function onLayout() { console.log('layout done'); } // bind event listener pckry.on( 'layoutComplete', onLayout ); // un-bind event listener pckry.off( 'layoutComplete', onLayout ); // return true to trigger an event listener just once pckry.on( 'layoutComplete', function() { console.log('layout done, just this one time'); return true; }); ```

dragItemPositioned

Triggered after a drag end, only when an item is positioned, either into a row or column, or into its container element. This event is triggered after the positioning transition has ended.

``` js .on( 'dragItemPositioned', function( pckryInstance, item ) { //... ```
  • pckryInstance

    Type: Packery

    The Packery instance

  • item

    Type: Packery.Item

    The item that was positioned

``` js // set up Packery and Draggabillies // like the bindDraggabillyEvents demo, then... pckry.on( 'dragItemPositioned', function( pckryInstance, item ) { console.log('Packery #' + pckryInstance.element.id + ' positioned ' + item.element.nodeName ); }); ```
