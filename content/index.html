Install

If you're not familiar with the command line, grab a packaged source file:

  • packery.pkgd.min.js for production
  • packery.pkgd.js for development

Bower

If you are cool with CLI, Packery can be installed with Bower. Packery is built on dependencies. Bower takes care of these.

``` bower install packery ```

Commercial licensing

Packery may be used in commercial projects with the one-time purchase of a commercial license. The license is priced per developer or organization. What is "commercial use" you ask? If you are paid to do your job, and part of your job is implementing Packery, a commercial license is required.

CSS

All sizing of items is handled by your CSS.

``` css .item { width: 25%; } ```

Initialize with JavaScript

Initialize a Packery instance with new Packery( element, options ). The Packery constructor accepts two arguments: the container element and an options object.

``` js var container = document.querySelector('#container'); var pckry = new Packery( container, { // options itemSelector: '.item' }); ```

Initialize in HTML

You can initialize Packery just with HTML, without writing any JavaScript. Simply add js-packery to the class of the container element. Options can be set with a data-packery-options attribute. Options must be valid JSON. Note that the attribute value uses single quotes ', but the JSON entities use double-quotes ".
