Install

If you're not familiar with the command line, just grab the packaged source files:

  • packery.pkgd.min.js for production
  • packery.pkgd.js for development

Bower

If you are cool with CLI, Packery can be installed with Bower. Packery is a library with dependencies. Bower takes care of these.

``` bower install packery ```

``` html ```

CSS

All sizing of items is handled by your CSS.

``` css .item { width: 25%; } ```

JavaScript

Initialize a Packery instance with new Packery( element, options ). The Packery constructor accepts two arguments.

  1. The container element
  2. An options object
``` js var container = document.querySelector('#container'); var pckry = new Packery( container, { // options itemSelector: '.item' }); ```
